freeCodeCamp: Image Search Abstraction Layer

Example of how to use this app:

fcc-image-abstraction-search.herokuapp.com/imagesearch/cats?offset=10

  • cats : enter your search terms here. Spaces ok!
  • /imagesearch/ ... ?offset= : always include as is.
  • 10 : changes amount of search results returned.

For a list of searched terms:

fcc-image-abstraction-search.herokuapp.com/history

Uses the IMGUR API so the imgs returned don't "exist" on a page -they are stored in "galleries."
